Right Click: Empty Space

 [CONTROL]
LMB & RMB: Place block
LMB & RMB + [shift]: Move player to cursor
[,] & [.]: Cycle LMB block type
[,] & [.] + [shift]: Cycle RMB block type
[-] & [+]: Alter level width
[-] & [+] + [shift]: Alter level height
[G]: Toggle god mode
[I]: Toggle info display
[C]: Toggle control display
[E]: Export level data
[E] + [shift]: Import level data
